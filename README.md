Project Title: Weather Forecast Application

Description
The Weather Forecast is a web application that provides real-time weather information and a 4-day forecast for a searched city or the user's current location. 
The project utilizes HTML, CSS, and JavaScript, along with external libraries such as Tailwind CSS and FontAwesome.

Features
- Search for weather by city name.
- Get weather data for the current location.
- Display current temperature, humidity, wind speed, and more.
- 4-day weather forecast visualization.

Setup Instructions
1. Clone the Repository
2. Navigate to the Project Directory
3. Open the index.html FileSimply double-click index.html or open it in your browser manually.

Usage
- Use the search bar to find the weather of a city.
- Click the location icon to get weather updates for your current location.
- View detailed weather information including temperature, humidity, wind speed, etc.
- Check the 4-day weather forecast.
